## Ejercicio 6 
```python
from functools import reduce
numeros = [[-3, 2, 7], [10, -5, 3], [0, 8, -2]]
listaPlana = reduce(lambda acum, item: acum + item, numeros,[])
listaPositiva = list(filter(lambda num: num > 0, listaPlana))
```
